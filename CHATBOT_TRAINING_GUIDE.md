## 🚀 Advanced Training Options

### Option 1: Use AI Services (OpenAI, Dialogflow)

**Using OpenAI ChatGPT:**
```bash
npm install openai
```

```javascript
import OpenAI from 'openai';

const openai = new OpenAI({
  apiKey: 'your-api-key',
  dangerouslyAllowBrowser: true
});

const getAIResponse = async (userMessage) => {
  const completion = await openai.chat.completions.create({
    model: "gpt-3.5-turbo",
    messages: [
      {
        role: "system",
        content: "You are an assistant for Shanruck Technologies training institute..."
      },
      { role: "user", content: userMessage }
    ]
  });
  return completion.choices[0].message.content;
};
```

### Option 2: Connect to Your Backend API

```javascript
const getBotResponse = async (userMessage) => {
  try {
    const response = await fetch('http://localhost:5000/api/chat', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ message: userMessage })
    });
    const data = await response.json();
    return data.response;
  } catch (error) {
    return "Sorry, I'm having trouble connecting. Please try our contact page!";
  }
};
```

## 🔧 Troubleshooting

**Chatbot icon not visible?**

- Clear browser cache (Ctrl + F5)
- Check browser console for errors

**Responses not working?**
- Verify the keyword matching in `getBotResponse`
- Check for typos in conditions
- Test with exact keywords first

**Styling issues?**
- Ensure Chatbot.css is properly imported
